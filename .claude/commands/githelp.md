Show git alias reference for two-repo workflow

| Command | Action |
|---------|--------|
| `goframework` | Navigate to Plugins (WWSimulatorFramework) |
| `gogame` | Navigate to game root (CPPDrugLordTycoonClone) |
| `pushframework` | Stage, commit, push framework only |
| `pushgame` | Stage, commit, push game only |
| `pushall` | Push both repos sequentially |
| `pullframework` | Pull framework only |
| `pullgame` | Pull game only |
| `pullall` | Pull both repos sequentially |

Run `/git` first to set up aliases for this session.
